# Mental Health Companion

This is a simple web-based mental health companion application.
It allows users to check their emotional state, track progress, and use a basic chat support system.

---

## Features

* Happiness survey with score calculation
* Chat support interface
* Progress tracking using charts
* Downloadable report (PDF)
* Light and dark mode
* Daily streak tracking

---

## Tech Stack

* HTML
* CSS
* JavaScript

---

## Project Structure

* `index.html` – Home page
* `login.html` – Login page
* `chat.html` – Chat page
* `survey.html` – Survey page
* `style.css` – Styling
* `script.js` – Functionality

---

## How to Run

1. Download or clone the repository
2. Open the folder
3. Run `index.html` in your browser

---

## Note

This is a frontend-only project.
All data is stored in the browser using localStorage.
